## Overview

Ziggy provides slash commands for configuration and status checks. These work in any Slack channel.

## Available Commands

### /ziggy connect

Connect your Jira Cloud instance to Ziggy.

```
/ziggy connect
```

Opens the Jira OAuth flow. Once connected, your 14-day free trial begins.

**Access:** All users

---

### /ziggy status

View connection and subscription status.

```
/ziggy status
```

Shows:
- Jira connection status
- Connected Jira site
- Default project
- Subscription status and trial days remaining

**Access:** All users

---

### /ziggy help

Show available commands and usage examples.

```
/ziggy help
```

**Access:** All users

## Configuring Projects

Use the **Home** tab in Slack to configure default projects, manage your subscription, and more. Open Ziggy in the Apps section of your Slack sidebar and click the **Home** tab.

## Command Reference

| Command | Description | Access |
|---------|-------------|--------|
| `/ziggy connect` | Connect Jira to workspace | All users |
| `/ziggy status` | View connection and subscription | All users |
| `/ziggy help` | Show help message | All users |
